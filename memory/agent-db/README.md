# Agent-Datenbank

Hier dokumentieren Agents ihre Aktionen und Learnings.
**Jeder Agent muss diesen Ordner VOR Aenderungen lesen.**

## Format

Jede Datei: `YYYY-MM-DD_<kurze-beschreibung>.md`

## Inhalt pro Eintrag

```
# Aktion: <was wurde gemacht>
## Dateien: <geaenderte Dateien>
## Learnings: <was man wissen muss>
## Risiken: <was koennte kaputt gehen>
```
